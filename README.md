# Owl Browser
一个简易的猫头鹰浏览器 

- 和猫头鹰没有任何关系
- 也不是只有晚上才能使用的
- 基于PyQt5.91 & Python3.6.5
- 版本迭代优化中
## 更新日志

v1.02

v1.07
