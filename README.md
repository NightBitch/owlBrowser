# Owl Browser
一个简易的猫头鹰浏览器 

- 和猫头鹰没有任何关系
- 也不是只有晚上才能使用的
- 基于PyQt5
- 版本迭代优化中
## 更新日志

v1.01 
- 基于py3.4和PyQt5.5实现基础功能

v1.02
- 加入书签功能，目前包括 百度, github 和 知乎 三个站点

v1.03
- 开启新的branch,增加基于py3.5和PyQt5.6的实现，对API进行相应变更

v1.04 
- 修复点击书签打开页面后，页面标题不同步更新的bug

v1.05
- 发布pyinstaller打包后的exe文件，并针对打包配置对代码进行更改

v1.06
- 地址栏字体优化，目前为微软雅黑
